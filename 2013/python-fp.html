Python 函数式编程笔记

2013-05-21 by atsuas

一直以来使用python的思维方式都是用以前写C时的那种思维，今天上网查了些资料，学习python的函数式编程方法，记录于此。

匿名函数

>>> f=lambda x:x+1
>>> f(5)
6

map(func,iter)

对于每个可遍历元素，调用func。返回包含func()结果的列表，例如：

>>> map(lambda x:x+1,[1,2,3])
[2,3,4]

filter(func,iter)

参数含义同map。返回func()结果为true的元素列表，例如：

>>> filter(lambda x:x%2,[1,2,3,4])
[1,3]

List comprehenstion

在一些需要使用循环的地方非常有用。例如：

#一重循环
>>> [x for x in [1,2,3,4] if x%2==1]
[1,3]
#二重循环
>>> [(x,y) for x in [1,2,3,4] for y in [0,1,2,3] if x+y==3]
[(1,2),(2,1),(3,0)]

闭包(closure)

绑定了外部作用域的变量(但不是全局变量)的函数，大部分情况下外部作用域是指外部函数。 例如求一个函数的导数，实现如下:

def d(f):
    def calc(x):
        dx=0.000001
        return (f(x+dx)-f(x))/dx
    return calc
